1.Can you create a Calendar object using the Calendar class?
Ans: no





2. Which method in the Calendar class is abstract?
Ans: add





3. How do you create a Calendar object for the current time?
Ans: no-arg constructor of class GregorianCalendar





4. For a Calendar object c, how do you get its year, month, date, hour, minute, and second?
Ans: c.get(Calendar.YEAR)
	c.get(Calendar.MONTH)
	c.get(Calendar.DAY_OF_MONTH)
	c.get(Calendar.HOUR)
	c.get(Calendar.MINUTE)
	c.get(Calendar.SECOND)
